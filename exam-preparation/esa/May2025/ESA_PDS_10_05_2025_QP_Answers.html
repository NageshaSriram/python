Q1 (a) How are dictionaries different from lists in Python?

Lists: Ordered collections of elements, accessed by index (0,1,2).

Dictionaries: Unordered collections of key-value pairs, accessed by keys.

In[34]:
# List
my_list = [10, 20, 30]
print(my_list[1])   # Output: 20

# Dictionary
my_dict = {"name": "Nagesha", "age": 25}
print(my_dict["name"])  # Output: Nagesha

# Difference: Lists use positions, dictionaries use keys.

Q1 (b) What is a lambda function? How is it different from a regular function?

Lambda function: Anonymous, single-line function defined using lambda.

Regular function: Defined using def, can have multiple statements.

Example:

In[35]:

Q3a. Write a Python program that takes a list of integers as input and returns a new list containing only the prime numbers that are also palindromes (e.g., 131, 7, 11). If no such numbers are found, return an empty list. (8 Marks)

In[45]:
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5)+1):
        if n % i == 0:
            return False
    return True

def is_palindrome(n):
    return str(n) == str(n)[::-1]

def prime_palindromes(nums):
    return [num for num in nums if is_prime(num) and is_palindrome(num)]

# Example
nums = [7, 11, 121, 131, 23, 101]
print(prime_palindromes(nums))  # Output: [7, 11, 131, 101]

[7, 11, 131, 101]

Explanation:

is_prime() checks primality.

is_palindrome() checks if number reads same forward/backward.

List comprehension filters numbers satisfying both.

students = {}

def add_student(name, marks):
    students[name] = marks

def update_marks(name, marks):
    if name in students:
        students[name] = marks
    else:
        print("Student not found!")

def average_marks():
    return sum(students.values()) / len(students) if students else 0

def display_students():
    for name, marks in students.items():
        print(f"{name}: {marks}")

# Example usage

If an employee has more than 10 years of experience, increase their bonus by 20% plus 3% of the base bonus multiplied by their performance rating.

If they have between 5 and 10 years of experience, increase their bonus by 15% plus 2% of the base bonus multiplied by their performance rating.

If they have less than 5 years of experience, increase their bonus by 10% plus 1% of the base bonus multiplied by their performance rating.

The function should return a new dictionary with employee names and their updated bonus amounts.

Example:

Input:

Output:

Adjusted salaries: Raju: $3960.0, Somu: $3025.0, Ravi: $2300.0, David: $5040.0

Problem: Adjust bonuses based on experience and performance rating.

Code:

In[47]:
def calculate_bonus(employees):
    updated = {}
    for name, (years, base_bonus, rating) in employees.items():
        if years > 10:
            bonus = base_bonus * (1.20 + 0.03 * rating)
        elif 5 <= years <= 10:
            bonus = base_bonus * (1.15 + 0.02 * rating)
        else:
            bonus = base_bonus * (1.10 + 0.01 * rating)
        updated[name] = round(bonus, 2)
    return updated

# Example
employees = {
    "John": (12, 10000, 4),
    "Mary": (7, 8000, 5),
    "Sam": (3, 5000, 2)
}
print(calculate_bonus(employees))

{'John': 13200.0, 'Mary': 10000.0, 'Sam': 5600.0}

Explanation:

Rules applied with conditional logic.

Multipliers adjust base bonus.

Rounded for neatness.

Output:

In[48]:
import re

def word_count(sentences):
    word_dict = {}
    for sentence in sentences:
        words = re.findall(r"[a-zA-Z]+", sentence.lower())
        for word in words:
            if len(word) >= 4:
                word_dict[word] = word_dict.get(word, 0) + 1
    return word_dict

# Example
sentences = ["Python is great!", "Programming in Python is fun.", "Fun with coding."]
print(word_count(sentences))

{'python': 2, 'great': 1, 'programming': 1, 'with': 1, 'coding': 1}

Explanation:

Regex extracts alphabetic words.

Converted to lowercase for case-insensitivity.

Dictionary counts words 4 characters.

from datetime import datetime

def analyze_dobs(names, dobs):
    dob_dict = {name: datetime.strptime(dob, "%Y-%m-%d") for name, dob in zip(names, dobs)}
    
    # Youngest student
    youngest = max(dob_dict, key=dob_dict.get)
    print("Youngest student:", youngest)
    
    # Students born in December
    december_students = [name for name, dob in dob_dict.items() if dob.month == 12]
    print("Born in December:", december_students)

# Example
names = ["Alice", "Bob", "Charlie"]
dobs = ["2000-12-15", "1999-05-20", "2001-12-01"]
analyze_dobs(names, dobs)

Youngest student: Charlie
Born in December: ['Alice', 'Charlie']

Explanation:

datetime.strptime parses DOBs.

max() finds latest date youngest.

Month check (dob.month == 12) filters December births.
